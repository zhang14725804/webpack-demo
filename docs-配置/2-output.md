### 2、output

output 配置如何输出最终想要的代码。output 是一个 object，里面包含一系列配置项

- (1) filename
- (2) chunkFilename
- (3) path
- (4) publicPath(加载异步资源对应的URL地址) 
- (5) crossOriginLoading(JSONP异步加载代码块) 

这些几乎没听说过

- (6) libraryTarget 和 library(构建一个可以被其他模块导入使用的库时需要用到它们) 
- (7) var (编写的库将通过 var 被赋值给通过 library 指定名称的变量) 
- (8) commonjs
- (9) commonjs2
- (10) this
- (11) window
- (12) global
- (13) libraryExport
